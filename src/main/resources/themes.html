Python

Рівень: Початковий

Примітки: Популярна у Data Science, веб-розробці, автоматизації.

Уроки:

  • Синтаксис та базові конструкції: змінні, оператори, умови, цикли.
  • Функції та модулі: створення та використання функцій, підключення модулів, робота з файлами.

Типи завдань:

  • Теоретичні завдання
  • Практичні вправи
  • Міні-проєкти

Java

Рівень: Середній

Примітки: Кросплатформенність через JVM, використовується у корпоративній розробці, Android-додатках.

Уроки:

  • Основи ООП: класи, об’єкти, наслідування, інкапсуляція, поліморфізм.
  • Колекції: ArrayList, HashMap, TreeSet, обробка даних у структурах.

Типи завдань:

  • Практичні завдання
  • Проєктні завдання

JavaScript

Рівень: Початковий

Примітки: Використовується для фронтенд-розробки, створення інтерактивних веб-сторінок.

Уроки:

  • Веб-розробка: HTML, CSS, інтеграція JS
  • DOM та події: маніпуляції елементами сторінки, обробка кліків і подій
  • Динамічні сторінки: робота з API, асинхронні запити

Типи завдань:

  • Практичні інтерактивні вправи
  • Міні-проєкти

C++

Рівень: Поглиблений

Примітки: Використовується для системного програмування, ігор, високопродуктивних додатків.

Уроки:

  • Системне програмування: управління пам’яттю, указівники, робота з файлами.
  • STL: вектори, списки, множини, алгоритми STL.

Типи завдань:

  • Алгоритмічні задачі
  • Практичні завдання високої складності

Типи завдань:

  • Практичні вправи
  • Міні-проєкти
